1.共模攻击

两个及以上的n来加密同一条信息m

c1=me1%nc2=me2%n\begin{aligned} c1 = m^{e1} \% n\\ c2 = m^{e2} \% n \end{aligned} c1=me1%nc2=me2%n​

假设e1，e2互质

gcd(e1,e2)=1\begin{aligned} gcd(e1, e2) = 1 \end{aligned} gcd(e1,e2)=1​

裴蜀定理：

    对于整数a、b存在整数x、y使得 gcd(a, b)== ax+by

    整数a、b互质的充分必要条件是存在整数 x、y 使得ax+by=1

扩展欧几里得 求逆元：

ax+by=1a ∗ x≡1 (mod b)\begin{aligned} ax + by &= 1 \\ a \ * \ x &≡ 1 \ (mod \ b) \end{aligned} ax+bya ∗ x​=1≡1 (mod b)​

根据上面已知a, x, b，再求y，所以根据扩展欧几里得算法得到：

e1∗s1+e2∗s2=1\begin{aligned} e1 * s1 + e2 * s2 = 1 \end{aligned} e1∗s1+e2∗s2=1​

s1、s2皆为整数，但是一正一负，假设s1为正数,s2为负数

幂运算是一种关于幂的数学运算。同底数幂相乘，底数不变，指数相加。同底数幂相除，底数不变，指数相减。幂的乘方，底数不变，指数相乘。

(a∗b)%n=(a%n ∗ b%n) % n(a%n)b % n=ab%n\begin{aligned} (a * b) \% n &= (a \% n \ * \ b \% n) \ \% \ n\\ (a \% n) ^ b \ \% \ n &= a ^ b \% n \end{aligned} (a∗b)%n(a%n)b % n​=(a%n ∗ b%n) % n=ab%n​


证明过程：

(c1s1∗c2s2)%n=[(me1%n)s1 ∗ (me2%n)s2] % n=[(me1%n)]s1%n ∗ [(me2%n)%n]s2 % n#(a∗b)%n=(a%n ∗ b%n) % n=(me1∗s1%n ∗ me1∗s2%n) % n#(a%n)b % n=ab%n=(me1∗s1 ∗ me1∗s2) % n#(a%n ∗ b%n) % n=(a∗b)%n=(me1∗s1+e1∗s2) % n#同底数幂相乘，底数不变，指数相加=m1%n#e1∗s1+e2∗s2=1=m%n#(m<n)=m\begin{aligned} (c1^{s1} * c2^{s2}) \% n &= [(m^{e1}\% n)^{s1} \ * \ (m^{e2}\% n)^{s2}] \ \% \ n\\ &=[(m^{e1}\%n)]^{s1} \%n \ * \ [(m^{e2}\%n) \%n] ^{s2} \ \% \ n \qquad \#(a * b) \% n = (a \% n \ * \ b \% n) \ \% \ n \\ &=(m^{e1 * s1} \%n \ * \ m^{e1 * s2} \%n) \ \% \ n \qquad \#(a \% n) ^ b \ \% \ n = a ^ b \% n\\ &=(m^{e1 * s1} \ * \ m^{e1 * s2}) \ \% \ n \qquad \#(a \% n \ * \ b \% n) \ \% \ n = (a * b) \% n\\ &=(m^{e1*s1 + e1 * s2}) \ \% \ n \qquad \#同底数幂相乘，底数不变，指数相加 \\ &=m^1 \% n \qquad \# e1*s1+e2*s2=1 \\ &=m \% n \qquad \# (m < n) \\ &=m \end{aligned} (c1s1∗c2s2)%n​=[(me1%n)s1 ∗ (me2%n)s2] % n=[(me1%n)]s1%n ∗ [(me2%n)%n]s2 % n#(a∗b)%n=(a%n ∗ b%n) % n=(me1∗s1%n ∗ me1∗s2%n) % n#(a%n)b % n=ab%n=(me1∗s1 ∗ me1∗s2) % n#(a%n ∗ b%n) % n=(a∗b)%n=(me1∗s1+e1∗s2) % n#同底数幂相乘，底数不变，指数相加=m1%n#e1∗s1+e2∗s2=1=m%n#(m<n)=m​

所以：

(c1s1∗c2s2%n)=m\begin{aligned} (c1^{s1} * c2^{s2} \% n) = m \end{aligned} (c1s1∗c2s2%n)=m​

只需要利用扩展欧几里得算法得到s1和s2，就可以得到m了


注意：

e1和e2也可以不互质，最后会使用gmpy2.iroot()来开方得到m，比如：[SWPUCTF 2021 新生赛]crypto1

3.低加密指数攻击

加密指数指的是e，e一般选取65535，当e很小，可以直接破解。

这类攻击在CTF题中，一般是 e = 3

如果e = 3，且 m ^ e < n，c开3次根式，得到m。

如果e = 3，且 m ^ e > n，那么设k，有：c = m ^ e + kn

爆破k，如果 c - kn能开三次根式，得到m。
